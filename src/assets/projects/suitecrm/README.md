# Castells Immerscom – English Language Learning Platform

## Project Overview

Castells Immerscom is a comprehensive English language learning platform developed for a well-established English academy in Barcelona, Spain. The website enables students of all age groups to explore language courses, prepare for Cambridge examinations, enroll in online programs, book private lessons, access study abroad opportunities, and request translation and interpretation services. The platform combines educational content with a modern, user-friendly experience to simplify course discovery and student engagement.

## Key Features

* Multi-category English courses (Children, Teenagers, Adults, Professionals)
* Online learning platform with business and general English courses
* Cambridge exam preparation courses (B2, CAE, Proficiency)
* Private English coaching and one-to-one learning sessions
* Study abroad programs and language immersion opportunities
* Translation and interpretation service enquiries
* Course catalog with detailed descriptions and pricing
* Student testimonials and success stories
* Blog and educational resources for English learners
* Newsletter subscription and contact enquiry forms
* Responsive, multilingual, and SEO-optimized website

## Technologies Used

**Frontend:** React.js, TypeScript, Tailwind CSS
**Build Tool:** Vite
**Backend:** Node.js
**Database:** Supabase (PostgreSQL)

## Project Summary

Castells Immerscom is a modern educational platform that provides students with a complete digital experience for learning English. From exploring language courses and preparing for internationally recognized Cambridge exams to booking private lessons and study abroad programs, the platform centralizes all educational services in one place. With an intuitive interface, multilingual support, responsive design, and integrated enquiry system, the website enhances student engagement while simplifying course management and enrollment for the institution.
